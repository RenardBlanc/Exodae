# Projet Exodae
## Développement d’outils de classification de profils d’aile et d’optimisation de leur performance à l’aide de Machine Learning.

__Auteur__ : Ilyas Baktache 
__Année__ : 2022

## Objectif du projet :
* Évaluer le potentiel du Machine Learning pour accélérer les calculs de performance de profil d’avion
* Analyser les données de profils d’avion existants.
* Développer des outils de classification de profil avec des algorithmes d’apprentissage non-supervisés.
* Développer des outils de génération de profils spécifiques.
